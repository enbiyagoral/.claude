# Proactive memory

- Always save user corrections and feedback to memory immediately — both what to avoid AND what worked
- After completing a task, ask yourself: "Did I learn anything reusable?" If yes, save it
- Prefer updating existing memory files over creating new ones — check MEMORY.md first
- Keep memories actionable: what you learned + how to apply it next time
- When discovering how something works in this project, save it as project memory
- Enforcement: Stop hook will remind you at session end, but save during the session when possible
