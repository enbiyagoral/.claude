# Archive

Outdated documentation kept for historical reference. This directory is listed
in `.claudeignore` — Claude never loads these files automatically (0 token cost).

Move docs here instead of deleting them. If you need Claude to reference
an archived file, explicitly ask: "read docs/archive/filename.md"

## What belongs here

- Superseded architecture decisions
- Deprecated API docs
- Old runbooks replaced by newer versions
- Post-mortems older than 6 months
- Migration guides for completed migrations

## What does NOT belong here

- Anything actively referenced by CLAUDE.md or skills (move it to the right place first)
- Secrets, credentials, or sensitive data (delete these, don't archive)
