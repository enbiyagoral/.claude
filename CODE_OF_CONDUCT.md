# Code of Conduct

This project is committed to a respectful, inclusive, and constructive collaboration environment.

## Our standards

We expect all participants to:

1. Be respectful and professional in all interactions.
2. Assume good intent and ask clarifying questions before escalating.
3. Give actionable feedback focused on code, docs, and behavior, not people.
4. Welcome contributors with different backgrounds and experience levels.
5. Keep discussions relevant to improving the project.

## Unacceptable behavior

The following behavior is not acceptable:

1. Harassment, discrimination, or hateful language.
2. Personal attacks, insults, or intimidation.
3. Deliberate disruption of discussions, reviews, or community spaces.
4. Sharing private information without explicit permission.
5. Sexualized language or unwelcome sexual attention.

## Scope

This Code of Conduct applies in all project spaces, including:

1. GitHub issues and pull requests.
2. Discussions and review threads.
3. Any public or private communication channels used for project collaboration.

## Reporting

If you experience or witness unacceptable behavior, report it via a private GitHub security advisory or contact the maintainer directly through GitHub profile channels.

When reporting, include:

1. What happened.
2. Where it happened.
3. Who was involved.
4. Any evidence (screenshots, links, logs).

## Enforcement

Maintainers may take any action they consider appropriate, including:

1. Warning the contributor.
2. Removing offensive content.
3. Temporarily restricting participation.
4. Permanent removal from project spaces.

Enforcement decisions are made to protect community safety and project health.
